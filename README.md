# Nikhilanand Arya — Personal Portfolio

This repository contains a single-page personal portfolio/academic homepage for Dr. Nikhilanand Arya.

## Files
- `index.html` — single-file responsive webpage (HTML + CSS).
- `assets/profile.jpg` — profile photo used on the page.

## Deployment
This site is set up to be hosted on GitHub Pages. You can either:
- Use the repository's `main` branch (Pages → Build from `main` / `/`), or
- Enable automatic deployment using the included GitHub Actions workflow.
