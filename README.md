# Nikhilanand Arya — Portfolio (Next.js)


This is a complete, runnable Next.js project created from your provided `index.html` + `styles.css`.


How to run:


1. Ensure you have Node.js 18+ installed.
2. Copy the files into a project folder (or `git clone` if added to a repo).
3. Run `npm install`.
4. Run `npm run dev` and open http://localhost:3000.


Notes:
- `public/assets/profile.jpg` is a placeholder — replace with your real photo.
- There's a simple API route at `/api/contact` that accepts POST JSON for contact form integration.
